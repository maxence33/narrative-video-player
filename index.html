Setup

Setting up the nvp player is fairly easy, just add the js and css to your webpage and an HTML Div with a unique Id.

Then add the following Js script bit:

			var options = {
			ContainerId: "myvid",
			images: [
				{"url": "./example/montagne.jpg", "start": "00:00", "title":"Beautiful mountains of New Zealand"},
				{"url": "./example/nuages.jpg", "start": "00:07", "title": "Blanket of clouds"},
				{"url": "./example/voitures.jpg", "start": "00:14", "title": "Car factory Wow !"},
				{"url": "./example/buildings.jpg", "start": "00:22", "title": "Buildings seen from the top"}	
				],
			videoSource: "./example/video.mp4",
			textDisplay: "title", 
					
		};

ContainerId is the Id of the Div you have just created

Images is an array of objects containing three values : the source of the image (or Url), the exact time this image will start the video at and a short title that wil be seen when your mouse will hover this image.

videoSource is the place where is located your video (pure video URL, Youtube or others not yet supported)

textDisplay is an indicator of the title should be displayed on hover or not. Values accepted are "title" or "no".
